OutKast - Git Up, Git Out

美國南岸饒舌雙人組合OutKast的這首Git Up, Git Out想要傳達的訊息就是在叫大家「要振作起來，不要虛度光陰」。

從充滿「故事性」的歌詞中說出了種種糜爛的生活型態，並在副歌講了這幾句

“ you need to git up, git out and git something
Don't let the days of your life pass by
Don't spend all your time tryna get high
How will you make it if you never even try?
Cause you and I got to do for you and I ”
雖然通往美好人生的路是非常艱難的，但是我們不能半途而廢，放棄自己追求夢想的權利，You need to git up, git out and git something。
